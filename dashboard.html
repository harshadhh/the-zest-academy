First-time setup — do this once
1Open your Google Sheet → click Extensions → Apps Script
2Delete any existing code. Paste the entire contents of Code.gs from the zip. Save (Ctrl+S).
3In Apps Script → click the gear icon (⚙) → Project Settings → Script Properties → Add property: key = sheet_id, value = your Sheet ID (from the Google Sheet URL)
4Click Deploy → New Deployment. Type = Web App. Execute as = Me. Access = Anyone. Click Deploy. Copy the URL.
5Paste the Script URL and Sheet ID in the fields above → click Save & Connect.
6Click 📤 Push Local → Sheets below to upload all existing data.
