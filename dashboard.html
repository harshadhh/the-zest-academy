First-time setup (do this once)
1Open script.google.com → New Project → paste Code.gs from the zip
2Click Deploy → New Deployment → Web App. Set Execute as: Me, Access: Anyone. Copy the URL.
3Create a Google Sheet, copy its ID from the URL (long string between /d/ and /edit), paste in Script Properties as sheet_id.
4Paste the Web App URL below and click Connect.
